# WorldEngine

Design specification for a **persistent world** platform: multi-scene narrative play with characters that can hold private and shared memory, occupy locations, communicate under explicit scopes, and—when authorized—perform real-world actions under operator approval.

This repository contains **documentation only**. It is not a SillyTavern fork or extension.

## Documentation

All specifications live under [`docs/`](docs/README.md). Start with the docs index for reading order and glossary.

## Status

v1 design spec — concepts extracted from a prior SillyTavern deployment and reframed for a greenfield implementation. See [`docs/appendix-provenance.md`](docs/appendix-provenance.md) for source mapping.
